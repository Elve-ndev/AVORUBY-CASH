# AVORUBY-CASH
Scoring de crédit agricole par IA multimodale. Face au stress hydrique au Maroc, ce moteur de décision croise données IoT, météo NASA et imagerie Sentinel-2 pour générer un score de résilience. Objectif : transformer la donnée technique en preuve financière pour débloquer le crédit (Avocats/Fruits Rouges).
